# CareerMentor Client Documentation

## Overview

CareerMentor.AI is a modern, AI-powered career development platform. This client folder contains the frontend code, built with React, Tailwind CSS, and Framer Motion, providing a visually engaging and interactive user experience.

---

## What This Website Can Do

- **AI Resume Optimizer:** Upload your resume and receive instant, actionable feedback for ATS optimization, keyword density, and formatting improvements.
- **Smart Interview Coach:** Practice with AI-driven mock interviews tailored to your chosen job role, with real-time feedback and coaching.
- **Career Roadmap AI:** Get a personalized learning and career advancement path, including skills assessment and milestone tracking.
- **24/7 Career Mentor:** Access instant, AI-powered career advice and answers to your questions at any time.
- **Smart Job Tracker:** Organize and track your job applications, receive analytics, reminders, and market insights.
- **User Authentication:** Secure sign up, login, and personalized dashboard for every user.
- **Animated, Responsive UI:** Enjoy a seamless, modern interface with smooth transitions and engaging visuals.

---

## Main Features

- **Landing Page:** Animated, gradient-rich homepage introducing the platform and its features.
- **InterviewBot:** Interactive AI interview simulation with feedback and coaching.
- **Authentication:** Secure login and registration pages.
- **Dashboard:** Personalized user dashboard (not shown here) for managing your career tools.
- **About Page:** Information about the platform and its mission.

---

## Folder Structure

- **/src**
  - **/assets**: Images and static files
  - **/components**: Reusable UI components
  - **/context**: React context providers (e.g., AuthContext)
  - **/pages**: Main page components (LandingPage, InterviewBot, etc.)
  - **/utils**: Utility functions
  - **App.jsx**: Main app component, sets up routes
  - **index.js**: Entry point

---

## Technologies Used

- **React**: UI library for building components
- **Tailwind CSS**: Utility-first CSS for rapid styling
- **Framer Motion**: Declarative animations and transitions
- **React Icons**: Icon library
- **Axios**: HTTP requests to backend API

---

## Running the Client

1. Install dependencies:
   ```
   npm install
   ```
2. Start the development server:
   ```
   npm start
   ```
3. Open [http://localhost:3000](http://localhost:3000) in your browser.
